# habitum_GUI

## Dev Setup

You'll need to running a habitum backend service (such as [https://github.com/Rausch-Labs/habitum_graphql](https://github.com/Rausch-Labs/habitum_graphql)) in dev mode.

Then just run:
```
yarn
yarn start-react
```
